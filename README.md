# Campus-ShuttleBus-Noti (CSN)
캠퍼스 교내 셔틀버스의 결행 정보를 메신저를 통해 안내해 주는 서비스 입니다.

[📝 개발 동기](https://l22hs.tistory.com/28)

## 🖥 개발 환경
- **Language :** Python 3.10
- **Framework :** BeautifulSoup, Request, pymysql
- **DBMS :** MySQL
- **API :** Telegram Bot API
